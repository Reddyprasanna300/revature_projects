Ping Project
Description
The project aims to provide the below functionalities:

Sign-up
 * Create Profile
Sign-in
 * Edit Profile
 * Delete Profile
 * View Profile
 * Search Profile
 * Sorting Profile
 * ViewALl Profile
Software required
 * Java 1.8
 * JDBC
 * Eclipse
 * Oracle SQL
